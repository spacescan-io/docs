# CAT

## Get CAT(Chia Asset Token) info

Retrieves NFT collection info by `asset_id`

```
https://api.spacescan.io/cat/info
    ?authkey=your_auth_key
    &asset_id=509deafe3cd8bbfbb9ccce1d930e3d7b57b40c964fa33379b18d628175eb7a8f
```
